## Project Goal
[cite_start]To build and manage a complete enterprise cloud environment using a Microsoft 365 E5 trial, focusing on secure identity management and automated device deployment[cite: 46].

## Technical Implementation
* [cite_start]**Identity & Access:** Configured **Microsoft Entra ID (Azure AD)** for user onboarding/offboarding and implemented **MFA (Multi-Factor Authentication)** to secure account access[cite: 47, 48].
* [cite_start]**Modern Management:** Deployed **Windows Autopilot** for zero-touch provisioning and managed mobile devices via **Intune (MDM)**[cite: 50].
* [cite_start]**Resource Administration:** Set up shared mailboxes, security groups, and distribution lists to streamline team communication[cite: 49, 50].
* [cite_start]**Security Policies:** Enforced device configuration policies and managed application deployments to ensure a secure corporate posture[cite: 50].

## Key Tools Used
* Microsoft 365 Admin Center
* Microsoft Entra ID
* Microsoft Intune
* Windows Autopilot
